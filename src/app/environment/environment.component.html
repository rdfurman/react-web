Hardware

This site is being served by an old machine that I keep running as a hobby server. Cost is low (electricity) and the added benefit is a learning experience for me.

I also happen to be lucky enough to own a condo in a new complex with fibre to the home (FTTH) provided by Shaw. This gives me a very respectable 150 Mbps down and 15 Mbps up, though I can scale this up to 1000 Mbps down and 50 Mbps up should I desire.

Software

I have leveraged a number of different languages, frameworks, libraries, and engines to make the various projects on this server possible.

Over the years I have learned a lot about Linux and running a machine exclusively from a command line (Ubuntu Server, Minecraft Server), building games using a common engine (Unity), and developing modern websites (Angular, Bootstrap, etc.).
